# Bitget(비트겟) 거래소 가입 절차 및 인증 가이드 (2025년 기준)

### 단계별 가입 안내
   
1. **가입 정보 입력**  
   - **이메일·휴대폰 번호 중 선택**하여 입력  
   - 비밀번호 조건: **8–32자, 숫자/특수문자/대문자 포함**
   ![가입 정보 입력](가입-정보-입력.jpg)
2. **추천코드 확인(중요)**  
   - 가입란의 **Referral Code** 입력칸 자동 입력 여부 확인  
   - 비어 있다면 **UD0PT2** 직접 입력  
   - 또는 링크를 다시 열어 자동 입력 상태 확인

3. **인증코드 입력**  
   - 이메일/SMS로 발송된 **6자리 인증코드**를 입력

   <!-- IMG: bitget-step3-code.jpg  
        👉 인증코드 입력창 화면
   -->
   ![인증코드 입력창](인증코드-입력창.jpg)

6. **보안 설정(권장)**  
   - 가입 완료 후 프로필 → **Security** 메뉴에서  
     **2단계 인증(2FA)** 활성화 추천

> **문제 해결 팁**  
> - 추천코드가 비어 있으면 쿠키를 초기화하거나 시크릿모드 해제 후 다시 접속  
> - 앱이 강제로 열릴 경우 팝업을 닫고 **“웹에서 계속하기”** 선택

## 2️⃣ 본인 인증(KYC) 절차

계정 개설 후 **KYC 인증**을 완료해야 출금 및 주요 기능 이용이 가능합니다.

1. 로그인 후 **Profile → Identity Verification** 선택  
2. **거주 국가 선택**  
3. **신분증 종류 선택** (주민등록증/운전면허증/여권)  
4. **신분증 앞·뒷면 업로드**  
5. **얼굴(셀피) 인증 진행**  
6. 검토 완료 후 **Verified** 상태 표시
